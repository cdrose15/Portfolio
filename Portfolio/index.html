Some of My Recent Work...

CarFinder

Created using Web API and AngularJS, this application gives you access to a database of over 60,000 different vehicles. You are able to sort through and narrow down your search by selecting a year, make, model, and trim. Once you have found something of interest you can then view more details, an image (Bing Search API), and recall information (NHTSA API) of your selected vehicle!

Demo Here

Personal Blog

This blog has been created using MVC, Razor, LINQ, HTML, Bootstrap, and jQuery. The database has been constructed using the Code-First model and I have implemented individual user authentication. The main page will show any user a small summary of each Blog post as well as a link to access all details from each post. If you would like to leave comments you must either Register or use the implemented 3rd Party Logins which include Facebook, Google, Microsoft, GitHub, Twitter, and LinkedIn.

Read Here

Personal Budget System

Created using MVC, Entity Framework, LINQ, SQL, and Razor, this application is a fully secured functional personal budget system. Using a Code First Database, this project tracks your inputted Accounts, Transactions, and Budgets to help you keep on track with your spending. With C# being the main programming language used, along with many custom controller actions using LINQ statements, this financial portal showcases many different progamming languages.
